# User Stories

## User Story 1
As a team leader, I want to assign tasks to team members so that work is organized and responsibilities are clear.

## User Story 2
As a team member, I want to view my assigned tasks so that I know what I need to work on.

## User Story 3
As a team member, I want to update the status of my tasks so that progress is visible to the team.

## User Story 4
As a team leader, I want to track team progress so that deadlines are met.

## User Story 5
As a team member, I want to communicate with my team so that issues can be resolved quickly.
